# README visual provenance

This directory contains the publish-ready visuals used by the multilingual README pages.

## Text-free banners

- `hero-banner.jpg` is a warm, light editorial illustration generated with built-in ImageGen. It shows the same project cat in idle, working, and sleeping poses, plus the cat riding the official OpenClaw mascot. The cat reference was identity-locked to the project's golden-shaded cat with a round face, short muzzle, green eyes, compact body, and calm expression.
- `relationship-banner.jpg` is a text-free functional illustration generated with built-in ImageGen. OpenClaw and the project cat exchange coral and blue signal paths with abstract terminal, web, and task symbols, communicating state events and desktop task input without embedding language-specific copy.
- Both banners intentionally contain no headings, labels, UI text, or generated product claims. They are shared by the English, Simplified Chinese, and Japanese README pages.

## Localized product visuals

- `config-console-en.jpg`, `config-console-zh-cn.jpg`, and `config-console-ja.jpg` are deterministic, sanitized console illustrations generated by `scripts/generate_readme_locales.py`. They use `$HOME/.openclaw`, contain no token, and do not expose a personal path or private configuration.
- `state-cycle-en.gif`, `state-cycle-zh-cn.gif`, and `state-cycle-ja.gif` are deterministic previews generated from `media/idle-core.mov`, `media/work-loop.mov`, and `media/nap-loop.mov`. Only the state labels change by locale; the cat animation is sourced from the application.
- The localized exports use installed system fonts selected through `fc-match`. Re-running the generator refreshes all six localized assets.

## Preserved source material

- `mascot-command.png` is the identity-preserving mascot composition with the four areas outside its rounded-square plate removed to transparency. The cat, OpenClaw mascot, lighting, and all pixels inside the rounded boundary remain sourced from the original project icon.
- `openclaw-icon.jpg` is an optimized export of the official OpenClaw macOS icon. It is not redrawn or AI-generated.
- `inspiration-real-cat.jpg` is cropped from the owner-supplied real-life cat photo. It was re-encoded without EXIF, GPS coordinates, device information, or capture time.
- `demo-task.gif`, `demo-text.gif`, and `demo-sleep.gif` were exported from real product recordings supplied by the project owner.
- `scene-quick-task.jpg`, `scene-subagent.jpg`, `scene-tools.jpg`, and `scene-result.jpg` are still frames from those real product recordings.

The source recordings and original personal photo are intentionally not copied into the repository. Only compressed, publish-ready exports are tracked.

## ImageGen prompts

### Text-free hero banner

Use case: ads-marketing. Asset type: text-free GitHub README hero banner, 16:9 landscape. Create a warm, light editorial scene using the exact project cat identity: a golden-shaded cat with a round face, short muzzle, large green eyes, compact body, and calm expression. Show several functional poses across one coherent composition: sitting idle, standing and working, holding a phone while busy, sleeping, and riding the official red OpenClaw mascot while pointing forward. Use warm ivory, pale apricot, soft coral, and restrained powder-blue accents with elegant spacing and subtle depth. No text, letters, labels, badges, UI copy, watermark, or invented product claim.

### Text-free relationship banner

Use case: infographic-diagram. Asset type: text-free GitHub README relationship banner. Create a warm, light, designer-crafted functional illustration. Place the official red OpenClaw mascot on the left and the exact project cat on the right. Connect them with a coral state-event flow toward the cat and a muted-blue task-input flow back toward OpenClaw. Add a few abstract, non-text symbols suggesting terminal work, web access, branching tasks, tool activity, and completion. Keep the layout clear, friendly, balanced, and editorial on an ivory background. No text, letters, labels, cards containing copy, UI screenshots, neon, dark cyberpunk styling, or watermark.

### Rounded mascot transparency

Use case: background-extraction. Change only the four square corner areas outside the existing rounded-square icon plate to a removable flat chroma-key background. Preserve the cat, OpenClaw mascot, pose, lighting, proportions, and all content inside the rounded boundary. The final tracked PNG uses the untouched original icon pixels with a deterministic antialiased alpha mask, preventing generative changes to the project cat.

## Export notes

- Banners, console illustrations, the real cat photo, and product stills are optimized JPEG files. The rounded project mascot is a PNG with alpha transparency.
- Motion previews are palette-optimized GIF files.
- Exact explanatory copy remains in Markdown or deterministic localized overlays, never in the generated banners.
